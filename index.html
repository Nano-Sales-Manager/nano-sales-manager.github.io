How It Works

1. Set Up Your Catalog

Add products with photos, set stock levels, min/max provisions, and reorder alerts. Define client clusters for tailored pricing and credits.

2. Create and Process Orders

Generate sales orders, add items, accept payments, and issue invoices or delivery notes via print or email. Update status to Completed.

3. Monitor and Report

Track the sales funnel, profitability, debtors, and inventory in real-time. Run reports on sales, purchases, finances, and stock needs.

4. Scale with Data

Import/export spreadsheets, handle purchases from suppliers, adjust taxes, and refine processes—all locally, no internet required.
